# Shamba Traceability

Open-source, EUDR-compliant agricultural traceability platform built on the Hedera network. Multi-commodity, blockchain-native, designed for cooperatives, processors, exporters, and EU importers who need to prove provenance from farm to shelf.

> Status: bootstrapping. The full scaffold lands in the first feature PR (`feat/scaffold-monorepo`). Detailed docs and an architecture overview will follow there.

A project of [Shamba Records Limited](https://shambarecords.com).
