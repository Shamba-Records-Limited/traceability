# Threat Models

Living threat models for Shamba Traceability. Each document follows the STRIDE framework where applicable; privacy threats use LINDDUN.

| Document                    | Scope                                                  |
| --------------------------- | ------------------------------------------------------ |
| `system-stride.md`          | End-to-end STRIDE pass over the platform               |
| `on-chain.md`               | Specific threats to on-chain state and key custody     |
| `dds-pipeline.md`           | DDS generation and TRACES NT submission                |
| `farmer-privacy-linddun.md` | LINDDUN privacy analysis with smallholder-farmer focus |

Threat models are revisited every release cycle and after any significant change to the architecture or trust boundaries.
